Description
================
Rest Client is a set of scripts that creates an easy to interact with a REST service. This library includes the ability to:
* JSON Parsing Built In
* Get, Post, Put, Delete Support built in
* Integrates with UnityWebRequest Network stack, and extensible to integrate with outher networking stacks
* Built in Error Handling Events 
* Simple to use, includes working example scene
* Handles Numerous security shemes.  Client Authentication, Bearer Authorization, or Custom Authorization
